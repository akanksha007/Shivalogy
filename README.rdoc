== Shivalogy

The Shivalogy is an ecommerce website currenlty in the development stage. Shivalogy has been developed using RubyonRails. The aim for implementing this is too understand and implement the best pratices of RubyonRails.

Things that have been implemented so far:

* Authentication - Devise gem have been used to implement authentication

* Email Setup - Mailcatcher gem is used to implement email of development environment.
